# AI Ordering SaaS Project

This is the editable project folder for the SaaS version of the AI phone ordering system.

Open this folder in VS Code when working on the SaaS product plan:

```text
C:\Users\pc\OneDrive\Desktop\vapi-call-sms-notify\ai-ordering-saas-project
```

## What this project is for

The goal is to build a self-service SaaS where businesses can:

- Sign up and log in.
- Add their own menu.
- Choose an AI voice.
- Edit guided AI instructions.
- Receive AI phone calls.
- Validate customer delivery addresses.
- Request M-Pesa payments.
- Top up a wallet.
- Pay for call usage automatically.

The business dashboard should hide the underlying providers. Businesses should not need to know about Vapi or Africa's Talking.

## Files in this folder

- `PROJECT_INDEX.md`: quick map of everything in the folder.
- `COSTS.md`: provider costs collected so far.
- `NEXT_STEPS.md`: build roadmap.
- `docs/SAAS_BLUEPRINT.md`: full SaaS blueprint.
- `docs/REQUIRED_ACCESS.md`: what access/secrets will be needed later.
- `database/001_saas_foundation.sql`: Supabase database foundation.

## Update rule

When we discover new costs, architecture decisions, or build steps, update this folder too. This folder is meant to stay readable and editable as the SaaS plan grows.
